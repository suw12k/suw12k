## Nouvelle
Esprit curieux.  
J'aime comprendre comment les systèmes fonctionnent réellement, du code de haut niveau au code de bas niveau.

Je me concentre principalement sur le développement bas niveau et la recherche en sécurité.  
Je crée des outils, j'analyse des fichiers binaires et je démonte des systèmes jusqu'à ce qu'ils soient compréhensibles.
